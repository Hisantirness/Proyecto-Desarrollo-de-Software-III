# Sistema de Gestión de Barbería

Proyecto final para la materia Desarrollo de Software III.

## Descripción
Sistema de reservas de citas para una barbería basado en arquitectura de microservicios.

## Tecnologías
- **Backend**: Java, Spring Boot (Web, Data JPA).
- **Frontend**: Thymeleaf.
- **Base de Datos**: MySQL.
- **Infraestructura**: Docker, Docker Compose.
